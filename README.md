## 你是伯乐 也是千里马

假如你往外求是：世有伯乐，然后又千里马。假如你往内求，便是：你是伯乐，也是千里马。

每个人都是一个航海家，而航海重要的三样东西，指南针、望远镜和食物，是你立足于海上的关键。当然你还需要一本海船的使用指南，一份天气预报和几名水手。

- 你是伯乐
- 你也是千里马
